4.1. Signup (Web UI)

자동결제 신청 (정기결제)

  • URI: HTTPS /stdwpay/std/u/v1/payauth/auto
  • HTTP Method: POST
  • 서비스 설명: 이미 등록된 특정 wpayToken을 자동결제용으로 암호화하여 등록합니다.
