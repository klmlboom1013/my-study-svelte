1.5. PIN Reset (Web UI)

결제 비밀번호 변경 (Web UI)

  • URI: HTTPS /stdwpay/std/u/v1/pinno/reset
  • HTTP Method: POST
  • 서비스 설명: WPAY 결제 PIN번호를 변경하는 서비스로, 휴대폰 본인인증 절차를 거친 후 결제 PIN 번호를 재설정한다

1. Request (form)

가맹점에서 이니시스 호출 시 사용하는 요청 전문입니다.

NameTypeReqLenEncDescURLsign
midStringO10가맹점 ID1
wpayUserKeyStringO100OWPAY 사용자 KEY [CI 1:1 매칭]2
ciString100본인인증 기관 사용자 고유 식별 값3
returnUrlStringO256Response Callback URLO4
signatureStringO100데이터 위변조 검증 값
resetCssUrlString160custom css 작성한 URLO
resetTextUrlString160custom text css 작성한 URLO
customTitleString256브라우저 상단 바 타이틀O
customShortcutUrlString256브라우저 상단 바 favicon 이미지 URLO

Request Signature 생성 Plain Text 순서: mid={value}&wpayUserKey={value}&ci={value}&returnUrl={value}&hashKey={발급받은 hashKey value}
