5.1. Payment Authentication (Web UI)

WPAY 결제 요청 – 표준 (Web UI)

  • URI: HTTPS /stdwpay/std/u/v1/payauth/stdMain
  • HTTP Method: POST
  • 서비스 설명: 가맹점에서 WPAY 표준 결제 인증창을 호출할 때 사용합니다.
