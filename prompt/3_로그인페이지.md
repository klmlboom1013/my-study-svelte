## UI 구성

- 서버 선택 영역
    - label: Server
    - Radio Button
        - values: src/lib/types/wpayServerType.ts 의 SERVER_TYPES
        - default value:
            - 빈 값으로 한다.
            - 브라우저 localStorage에 Server 값이 있으면 localStorage의 Server 값으로 초기화 한다.
        - 하나만 선택 가능.
        - Inner Section: Domain 선택 영역
            - label: Domain
            - PROD 선택 시 Inner Section: Domain 선택 영역이 추가 된다.
            - Modal 로 어떤 PROD 서버로 접속할 건지 선택해야 한다.
            - 선택은 Radio Button 기능으로 제공 한다.
                - values: src/lib/types/wpayServerType.ts 의 PROD_SERVER_DOMAINS
                - default value:
                    - 빈 값으로 한다.
                    - 브라우저 localStorage에 PROD Server 값이 있으면 localStorage의 PROD Server 값으로 초기화 한다.
                - 세로로 정렬 한다.
        - PROD 서버 선택 결과 노출
            - radio button PROD 텍스트 옆에 "(선택한 도메인)" 문구를 표시 한다.
            - 도메인이 선택되어 있을 때에도 radio button PROD를 선택하면 도메인을 변경하기위해 Modal이 다시 뜬다.
        - DEV, STG 서버 선택
            - PROD의 Inner Section Domain 값이 있으면 Inner Section Domain 값을 유지 하지 않는다.
            - Inner Section Domain 값을 빈 값으로 한다.
            - radio button PROD 텍스트 옆에 "(선택한 도메인)" 문구를 표시하지 않는다.
        - required: true
- 서비스 선택 영역
    - label: Service
    - Dropdown Box & text Input 기능 혼합.
        - Dropdown Box 로 원하는 서비스는 선택하거나 텍스트를 입력하여 찾는 것이 가능 하다.
        - values: src/lib/types/wpayServerType.ts 의 SERVICE_OPTIONS
        - default value:
            - 빈 값으로 하며 "선택해 주세요" 메시지를 표시 한다.
            - 브라우저 localStorage에 Service 값이 있으면 localStorage의 Service 값으로 초기화 한다.
    - required: true
- 사이트 선택 영역.
    - label: Site
    - Dropdown Box & text Input 기능 혼합.
        - Dropdown Box로 원하는 사이트는 선택 하거나 텍스트를 입력 하여 찾는 것이 가능 하다.
        - values: src/lib/types/wpayServerType.ts 의 SITE_OPTIONS
        - default value:
            - 빈 값으로 하며 "선택해 주세요" 메시지를 표시 한다.
            - 브라우저 localStorage에 Site 값이 있으면 localStorage의 Site 값으로 초기화 한다.
        - required: true
- Merchant ID 선택 영역.
    - label: Merchant ID
    - Dropdown Box & text Input 기능 혼합.
        - Dropdown Box 로 원하는 채널은 선택하거나 텍스트를 입력하여 찾는 것이 가능 하다.
        - values: src/lib/types/wpayServerType.ts 의 MERCHANT_ID_OPTIONS
        - default value:
            - 빈 값으로 하며 "선택해 주세요" 메시지를 표시 한다.
            - 브라우저 localStorage에 Merchant ID 값이 있으면 localStorage의 Merchant ID 값으로 초기화한다.
        - required: true
- Member ID 입력 영역.
    - label: Member ID
    - text Input
        - default value:
            - 빈 값으로 한다.
            - 브라우저 localStorage에 Member ID 값이 있으면 localStorage의 Member ID 값으로 초기화 한다.
        - hint: wpayTestUser01
        - focus in 시 hint 가 사라진다.
        - focus out 시 입력된 값이 없으면 hint 가 나타난다.
        - required: false
- Cell Phone Number 입력 영역.
    - label: Cell Phone Number
    - number Input
        - default value:
            - 빈 값으로한다.
            - 브라우저 localStorage에 Cell Phone Number 값이 있으면 localStorage의 Cell Phone Number 값으로 초기화 한다.
        - 복사한 문자열을 붙여 넣기 할 경우 숫자를 제외한 문자를 제거한다.
        - required: false
- Remember Me 입력 영역.
    - label: Remember Me
    - Check Box
        - default value:
            - 체크 박스를 체크하지 않는다.
            - 브라우저의 localStorage에 Remember Me 값이 있으면 localStorage의 Remember Me 값으로 초기화 한다.
        - required: false
- Next Button 영역.
    - 버튼 문구: 다음
    - 버튼 활성화 조건
        - Service, Server, Site, Merchant ID 값이 모두 선택 되었을 때
        - Click 이벤트 처리
            - 메인 화면으로 이동 한다.
            - remember me 체크 박스가 체크 된 상태라면 사용자가 선택한 데이터와 입력한 데이터들을 브라우저의 localStorage에 저장한다.
            - remember me 체크 박스가 체크 되지 않은 상태라면 localStorage에 저장된 데이터를 삭제한다.
            - Member ID 가 빈 값이면 "wpayTestUser01" 값을 강제로 세팅 한다.
    - 버튼 비활성화 조건
        - Service, Server, Site, Merchant ID 값 중 하나라도 선택 되지 않았을 때
        - Mouse Over
            - Service, Site, Merchant ID 의 테두리 색상을 Red 계열로 변경 한다.
            - Server는 radio button 의 색상을 Red 계열로 변경 한다.
            - 마우스가 Next Button 영역을 벗어나면 원래 스타일로 돌아온다.
        - Touch (모바일 환경)
            - Service, Site, Merchant ID 의 테두리 색상을 Red 계열로 변경 한다.
            - Server는 radio button 의 색상을 Red 계열로 변경 한다.
            - 2초 뒤 원래 스타일로 서서히(2초동안) 돌아온다.
