// turbo-all

This workflow executes the change language.

1. AI Agent 대화 메시지는 기본 한국어로 설정 한다.

2. AI Agent 가 작성한 "Task"를 한국어로 번역해서 보여주며 기본 언어 설정을 한국어로 한다.

3. AI Agent 가 작성한 "Walkthrough"를 한국어로 번역해서 보여주며 기본 언어 설정을 한국어로 한다.

4. AI Agent 가 작성한 "Implementation Plan"을 한국어로 번역해서 보여주며 기본 언어 설정을 한국어로 한다.
