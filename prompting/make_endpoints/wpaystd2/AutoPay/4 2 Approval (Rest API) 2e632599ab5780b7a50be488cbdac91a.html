4.2. Approval (Rest API)

자동결제 승인 – WPAYPLUS 자동결제 승인

  • URI: HTTPS /stdwpay/std/apis/payreqappl
  • HTTP Method: POST (Rest / Content-Type: application/json)
  • 도메인:

◦ [스테이지] https://stgwpayplus.inicis.com

◦ [운영서버] https://wpayplus.inicis.com

  • 서비스 설명: 자동결제 신청 응답으로 받은 wpayToken과 wtid를 활용하여 실제 승인을 처리합니다.

1. Request (Rest)

가맹점 서버에서 WPAY 서버로 승인을 요청하는 전문입니다.
